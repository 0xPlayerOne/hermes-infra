# Daily Standup

**Schedule:** Configure per profile.

**Prompt:**

Post one concise daily standup to `$STANDUP_CHANNEL` using the active profile's name and role.

Rules:
- Do not reply to other standups.
- Use mentions only when directing information to a specific profile.
- Keep the message under 300 words.
- Start with `<profile> — Daily Standup (date)`.

Format:
- What I worked on
- Insights for the team
- Known issues or blockers
- New tools or skills

Use recent sessions as evidence. Do not invent activity.
